How it works

Share the YouTube URL and tell us which product to focus on. We confirm scope within a few hours.

Our pipeline classifies each comment — separating product feedback from YouTuber opinions and noise. Every pain point gets attribution and severity.

A structured PDF with executive summary, ranked pain points with recommended actions, feature requests, competitive intelligence, and a prioritized action checklist.

03 — Open Source

Run it yourself

The core analysis engine is open source. Analyze up to 50 comments per video, get a basic HTML report. No sign-up, no server, no catch.

What the open-source edition includes

Sentiment classification, pain point extraction, feature request tagging, praise highlights — with support for Claude, OpenAI, Gemini, DeepSeek, and MiniMax. Everything runs locally on your machine.

The paid reports go further: executive summaries with counter-intuitive findings, attribution analysis per pain point, competitive intelligence, prioritized action checklists, and analysis of 500+ comments for statistical confidence.
